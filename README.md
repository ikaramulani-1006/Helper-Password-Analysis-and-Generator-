# Helper: Password Analysis and Generator

An advanced interactive Password Strength Analyzer built using Python and ipywidgets.

This tool provides real-time password analysis, crack-time estimation, password generation, password history tracking, dark mode UI, and export functionality.

## Features

- Real-time password strength analysis
- 12 advanced security checks
- Password entropy calculation
- Crack-time estimation
- Common password detection
- Keyboard pattern detection
- Secure password generator
- Copy password to clipboard
- Export analysis results as JSON
- Dark mode support
- Password history tracking
- Interactive GUI using ipywidgets

## Technologies Used

- Python
- ipywidgets
- Regular Expressions (re)
- JSON
- Math
- Datetime
- Pyperclip
